Open Source Kanji Writing Tool

Kanjify is an open-source kanji writing tool. Kanjify is written in Javascript and uses SVG from the excellent kanjiVG project.

Try it!

Animate the character: ÁßÅ



Why does this exist?

Stroke order animations are cumbersome to create and difficult to customize. Kanjify offers a customizable SVG animation of kanji from kanjiVG. Kanjify is written in Javascript so that it can work across as many platforms and devices as possible.
